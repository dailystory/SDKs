DailyStory Web Form Examples

To learn more about DailyStory's Web Forms, please visit Web Forms documentation and the DailyStory PHP SDK Web Forms APIs.

The examples included use an existing DailyStory application. If you would like to use these samples with your own DailyStory application or trial you will need to update the SiteId and Web Form Id values in each of the php pages.

Example Description
Basic Form Basic web form using the SDK.
Basic Form with CSS Adds DailyStory's base web form style sheet.
Basic Form Horiztonal with labels Uses CSS to show horizontal web form with labels.
Basic Form Horiztonal without labels Uses CSS to show horizontal web form with no labels (configured in the Web Form Designer).
Advanced CSS Form Uses custom CSS to display form and Bootstrap for page layout.
Custom HTML Form User defined HTML Form that posts to DailyStory.
